*******************************************************
*  Description of the program
*******************************************************
This program helps to find union and intersection of two sets in a new set where union contains combined contents of the original
two sets and the intersection contains only duplicate contents of the original two sets.

*******************************************************
*  Source files
*******************************************************

Name:  main.cpp
   Main program. This is the main program which creates a vector for two sets where 
it takes two sets of integers from an input file and store them into two separate sets and also
contains vector class which is a class template for sequence container. This program also combines two sets using union function 
and intersection function.

Name:  Sets.h
   Contains the definition of Header for Sets class.  

Name: function.cpp
   Contains the definition of Header and Constructur for Sets class. It also contains class template vector and union and intersection
function.
