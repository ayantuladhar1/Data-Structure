*******************************************************
*  Description of the program
*******************************************************
This program helps to simulate a cash wash and calculate the arrival time, departure time and wait time for each car that came in for a car wash.
Car wash simulation will run from 08:00 am till 5:00pm(9 hours or 540 minutes).




*******************************************************
*  Source files
*******************************************************

Name:  Source.cpp
   
   Main prohram. This is the main program which usues queue, vector, math data operation to perform car wash simulation.
This program contains string, push, pop, empty, push_back, float, floor and math operations to calculate and diplay the information
about each car such as car number, arrival time, car wash start time, departure time, wait time and total time in form of table.
This program also displays the following statistics such as total wait time, average wait time, total car wash use time and 
percentage of time car wash was in use.

Name:  Header1.h
   Contains the definition of Header for Car class.
