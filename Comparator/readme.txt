*******************************************************
*  Description of the program
*******************************************************

This program helps to define a rectangle class which creates an array of rectangles
and finds the largest rectangles first on the basis of area and then on the baqsis of perimeter.


*******************************************************
*  Source files
*******************************************************

Name:  main.cpp
   Main program.  This is the main program which creats a vector of five rectangles
and objects which includes findMax template to find the largestitem.

Name:  rectangle.h
   Contains the definition of Header for rectangle class.  

Name: rectangle.cpp
   Contains the definition of Header and Constructur for rectangle class.

Name: RectangleAreaComparator.h
   Contains the comparator class which compares rectangles by area.

Name: RectanglePerimeterComparator.h
  Contains the comparator class which compares rectangles by perimeter.
